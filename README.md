## API

如果你想以此mod为前置, 请在`build.gradle`里键入如下内容

```groovy
repositories {
	maven { url 'https://maven.cjsah.net/repository/releases/' }
}

dependencies {
	modImplementation "net.cjsah.cpp:craftingpp:{VERSION}"
}
```
